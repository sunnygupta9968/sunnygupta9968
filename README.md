## Featured Projects

### DropIt — Temporary File Sharing Platform

**Tech Stack:** Spring Boot, React.js, Redis, Docker, Render, Vercel

A full-stack temporary file-sharing platform that allows users to upload files and share them using short six-digit access codes.

**Key Features:**

- Code-based temporary file sharing
- Spring Boot REST APIs for upload and download
- Redis metadata expiry for automatic access invalidation
- Scheduled cleanup for expired files
- Deployed using Render and Vercel

**Live:** https://dropit-blush.vercel.app/  
**GitHub:** https://github.com/sunnygupta9968/dropit

---

### Alumni Management System

**Tech Stack:** JSP, Servlets, MySQL, JDBC, BCrypt

A JSP and Servlet-based alumni management platform with Admin and Alumni dashboards.

**Key Features:**

- Secure login and registration
- BCrypt password hashing
- Session management and role-based access control
- Alumni verification workflow
- Job posting and referral workflows
- Built during a 48-hour hackathon

**GitHub:** https://github.com/sunnygupta9968/alumni_connect

### Hospital Management System

A hospital management system with separate Admin, Doctor, and User modules.
